Comenzamos a configurar el Switch S1

Ingresamos a S1 y configuramos los puertos correspondientes para que la VLAN 1 funcione como una VLAN de voz o una VLAN de datos en ellos.

S1>enable
S1#configure terminal

// Permitimos que la VLAN 1 funcione como VLAN de voz en los puertos FastEthernet 0/1 y FastEthernet 0/2
S1(config)#interface range f0/1-2
S1(config-if)#switchport mode access
S1(config-if)#switchport voice vlan 1
S1(config-if)#exit

// Permitimos que la VLAN 1 funcione como VLAN de voz y de datos en el puerto FastEthernet 0/3
S1(config)#interface f0/3
S1(config-if)#switchport mode access
S1(config-if)#switchport access vlan 1
S1(config-if)#switchport voice vlan 1
S1(config-if)#exit

// Opcional, Guardamos la configuracion
S1(config)#exit
S1#write

Comenzamos a configurar R1

Configuramos el direccionamiento IP del router y creamos un pool DHCP, el cual brindara direcciones IP a todos los hosts de nuetra red, adicionalmente, dentro del DHCP configuraremos la opcion 150, el cual corresponde al servicio de TFTP.

R1>enable
R1#configure terminal
R1(config)#interface f0/1
R1(config-if)#ip address 192.168.10.1 255.255.255.0
R1(config-if)#no shutdown
R1(config-if)#exit

// Creamos el pool para DHCP, el nombre del DHCP que usare es RED_1, pero se puede colocar cualquiera
R1(config)#ip dhcp pool RED_1
R1(config)#network 192.168.10.0 255.255.255.0 (Direccion de red)
R1(config)#default-router 192.168.10.1 (Direccion del gateway)
R1(config)#dns-server 8.8.8.8 (Opcional, direccion del servidor DNS, usare el de google)
// Configuro la opcion 50 del DHCP, el servicio TFTP
R1(config)#option 150 ip 192.168.10.1 (usamos el gateway de la red)
R1(config)#exit

Configuramos el servicio telefónico Call Manager Express CME (Communications Manager Express) en R1.

R1>enable
R1#configure terminal

// Configurando el servicio telefonico CME
R1(config)#telephony-service

// Establecemos el número máximo de números
R1(config-telephony)#max-dn 3

// Establecemos el número máximo de teléfonos
R1(config-telephony)#max-ephones 3

// Indicamos cual será el Router que entregue la configuracion a los teléfonos IP (aqui es R1), incluyendo el puerto (default 2000).
R1(config-telephony)#ip source address 192.168.10.1 port 2000

// Asignamos automáticamente extensiones a los botones
R1(config-telephony)#auto assign 1 to 3

Despues de haber configurado el CME en R1, encendemos los telefonos VoIP 1, VoIP 2 y VoIP 3.

Una vez terminada la configuracion de los dispositivos de red, encendemos los telefonos VoIP 1 y VoIP 2, y configuramos la PC 1.
